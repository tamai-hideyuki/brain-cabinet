# 評価と再学習スケジュール

| 評価 | 意味 | 次回レビュー | 効果 |
|------|------|-------------|------|
| 5️⃣ | 完全に理解 | 間隔が大きく延びる | 復習頻度↓（得意なものは後回し） |
| 4️⃣ | ほぼ理解 | 間隔が少し延びる | 復習頻度やや↓ |
| 3️⃣ | 少し迷ったが理解 | 間隔が維持〜微増 | 現状維持の復習ペース |
| 2️⃣ | あまり理解できなかった | 1日後にリセット | すぐ再学習 |
| 1️⃣ | 全く思い出せなかった | 1日後にリセット | すぐ再学習 |
| 0️⃣ | 間違って理解していた | 1日後にリセット | すぐ再学習 |
